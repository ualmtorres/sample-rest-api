API Documentation

GET /products

Obtiene todos los productos disponibles.

Respuestas: 200 Lista de productos encontrada, 204 No hay productos disponibles
GET /products/{id}

Obtiene un producto específico por su ID.

Parámetros: id (path)
Respuestas: 200 Producto encontrado, 204 Producto no encontrado
POST /products

Crea un nuevo producto.

{ "name": "string", "price": "number" }
Respuestas: 201 Creado, 400 Error
PUT /products/{id}

Actualiza un producto existente.

Parámetros: id (path) 
{ "name": "string (opcional)", "price": "number (opcional)" }
Respuestas: 200 Actualizado, 204 No encontrado, 400 Error
DELETE /products/{id}

Elimina un producto existente.

Parámetros: id (path)
Respuestas: 200 Eliminado, 204 No encontrado, 400 Error
